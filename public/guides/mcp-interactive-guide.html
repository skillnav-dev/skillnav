Step 1 / 4

什么是 MCP？

Model Context Protocol — 让 AI 模型安全访问外部工具和数据的开放协议

🖥️ 宿主应用
Claude / Cursor / VS Code
⟷
📡 MCP 协议
标准化通信层
⟷
⚙️ MCP 服务器
GitHub / DB / 文件系统
💡 类比理解：MCP 就像 USB-C 接口 — 一个标准协议连接所有设备。无需为每个 AI + 工具组合编写专用代码。
🔌

即插即用

同一个 MCP 服务器可以被 Claude Desktop、Cursor、VS Code 等任何兼容客户端调用，配置格式统一。
🔒

安全可控

每次工具调用需用户授权，服务器通过 stdio 本地通信，不暴露给外部网络。
🧩

生态丰富

官方 + 社区已有 7000+ 个 MCP 服务器，覆盖数据库、API、文件系统、搜索、自动化等场景。
⚠️ 前置要求：需要安装 Node.js ≥ 18（用于 npx 运行服务器），以及对应的宿主应用。
