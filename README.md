SkillNav 帮助中文开发者发现、评估和使用 AI Agent 工具。我们持续追踪 AI 智能体生态，提供结构化的工具导航、每日精选资讯和深度技术专栏。

## 收录内容

- **168+ Skills** — Claude Code 自定义技能，按场景分类、安全评分
- **3,900+ MCP Servers** — 模型上下文协议服务器精选导航，S/A/B 分级
- **240+ 资讯** — 每日自动采集 10+ 海外 AI 源，中文编译发布
- **11 深度专栏** — AI 架构、Prompt 工程、RAG vs 微调等交互式指南
- **每日简报** — AI 热点 5 分钟速览，多平台同步分发

## 技术栈

Next.js 15 (App Router + RSC + ISR) · Tailwind CSS + shadcn/ui · Supabase (PostgreSQL + PGroonga 中文全文搜索) · Cloudflare Workers (OpenNext) · GitHub Actions CI/CD

## 本地开发

```bash
npm install
npm run dev    # http://localhost:3000
```
